# Audio Assets

Place your audio files in this directory:

1. van-gogh-slowed.mp3 - Dark theme music
2. moby-porcelain.mp3 - Light theme music

Audio requirements:
- Format: MP3
- Bitrate: 128-192 kbps
- Sample rate: 44.1 kHz
- Channels: Stereo
- Duration: Full songs
- File size: Optimize to under 5MB per file
